HTML는 무엇인가?

하이퍼텍스트 마크업 언어(HyperText Markup Language, HTML, 문화어: 초본문표식달기언어, 하이퍼본문표식달기언어)는 웹 페이지를 위한 지배적인 마크업 언어다. 또한, HTML은 제목, 단락, 목록 등과 같은 본문을 위한 구조적 의미를 나타내는 것뿐만 아니라 링크, 인용과 그 밖의 항목으로 구조적 문서를 만들 수 있는 방법을 제공한다. 그리고 이미지와 객체를 내장하고 대화형 양식을 생성하는 데 사용될 수 있다. HTML은 웹 페이지 콘텐츠 안의 꺾쇠 괄호에 둘러싸인 "태그"로 되어있는 HTML 요소 형태로 작성한다. HTML은 웹 브라우저와 같은 HTML 처리 장치의 행동에 영향을 주는 자바스크립트와 본문과 그 밖의 항목의 외관과 배치를 정의하는 CSS 같은 스크립트를 포함하거나 불러올 수 있다. HTML과 CSS 표준의 공동 책임자인 W3C는 명확하고 표상적인 마크업을 위하여 CSS의 사용을 권장한다.

HTML의 문법

출처 : https://www.advancedwebranking.com/html/#overview



※ 태그가 있다고 간주하고 적음
※다른 문서를 참고한것이 아니라 내 머릿속에 있는걸 적은거라 정의가 정확하지 않을 가능성이 높음

  1. html : html의 처음과 끝을 나타내는 것
  2. head : 주소의 제목(title), meta 등을 삽입하는 영역을 나타낸 것
  3. body : 웹페이지 안에 있는 내용을 삽입하는 영역을 나타낸 것
  4. title : 주소의 제목을 나타내기 위해 사용
  5. a : 다른 주소의 링크를 나타내기 위해 사용. 주로 href 옵션을 사용하여 주소를 가져옴
  6. img : 이미지를 가져오기 위해 사용. 주로 src 옵션을 사용하여 이미지를 가져옴
  7. p : 글의 문단을 나타낼 때 사용
  8. li : list로 나열하고자하는 내용이 있을 때 사용
  9. ul : li를 모아 하나로 뭉치거나 분류하고 싶을 때 사용
  10. ol : ul에서 리스트를 정리해주는 기능을 함
  11. br : 한줄 띄워주는 역할을 함
  12. h : 웹페이지의 제목을 나타냄. h1부터 h6까지 있으며 뒤의 숫자가 커질수록 크기가 작아짐
